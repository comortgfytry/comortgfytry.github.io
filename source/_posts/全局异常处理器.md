# 快速使用
```java
@RestControllerAdvice
public class globalExceptionHandler {
    @ExceptionHandler(Exception.class)  // 异常类型
    public Result ex(Exception ex){
        ex.printStackTrace();
        return Result.error("对不起，操作失败，请联系管理员");
    }
}
```
